MMETSP samples on imicrobe vs. NCGR¶

The SRA BioProject id for the MMETSP is: PRJNA231566 This contains 678 samples (MMETSP id in SampleName) and 719 SRA Experiments (Run). Note that there are 2 sub-projects, PRJNA248394 with 718 samples and PRJNA245816 with 1 extra Run from Tetraselmis sp. GSL018.

Files from imicrobe were downloaded from the ftp site.

ncbi<-read.csv("../SraRunInfo.csv")
ncbi_719<-read.csv("../SraRunInfo_719.csv")
source('../extra_scripts/overLapper.R') # plots venn diagrams

MMETSP_id_imicrobe<-imicrobe$SAMPLE_NAME
